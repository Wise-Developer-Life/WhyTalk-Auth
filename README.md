# WhyTalk-Auth
Authentication Service of WhyTalk Chatting App


### Spotless
1. Check coding style
	```shell
	./gradlew spotlessCheck
	```
2. Apply coding style
	```shell
	./gradlew spotlessApply
	```

### Swagger
1. Generate swagger.json in build/docs
	```shell
	./gradlew clean generateOpenApiDocs
	```
